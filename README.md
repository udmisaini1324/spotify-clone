# 🎵 Spotify Clone

A responsive Spotify-inspired music player built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🎵 Play / Pause songs
- ⏮ Previous & Next controls
- 📊 Progress bar with seek
- 🔊 Volume control
- 🔍 Live song search
- 🌅 Dynamic greeting
- 📱 Responsive design
- 🎨 Spotify-inspired UI
- ⚡ Smooth hover animations

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
